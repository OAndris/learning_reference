Debounce & Throttle example

Both debounce and throttle are used for delaying the firing of callbacks. They can improve user experience, reduce the number of unnecessary API calls and save both bandwidth and money.

Debounce use case examples: performing validation and/or sending API request only after the user has probably stopped typing.

Throttle use case examples: eager validation of longer inputs, and/or monitoring events like resize, scroll and mouse movement.
